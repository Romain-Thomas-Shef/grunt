Install from github
-------------------

To install from github you can:

* Clone the `repository <https://github.com/Romain-Thomas-Shef/grunt>`_ and create a local install from it
* Use pip to install it directly from github:

.. code-block:: shell

    pip install git+https://github.com/Romain-Thomas-Shef/grunt
