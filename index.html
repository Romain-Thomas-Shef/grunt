GRUNT: Garmin RUN Tracking

GRUNT is a very small tool that I made to track my running stats. I have used this mainly to populate this page. It is made using garmin running data that are downloaded by grunt.

Contribute: If you find a bug or would like to propose a new feature, I would appreciate if you could write an issue in the github repository. If you want to contribute to the development. If this is your case please have a look at the Development corner and if you have any question you can write to and R.Thomas (romain.alexis.gerard.thomas@gmail.com).
